# Windows セットアップ

## インストールが必要なアプリ

- エディタ＆IDE等
  - Visual Studio Code
  - EmEditor
  - Intellij IDEA（Jetbrains tool Box）
  - [GVim](https://www.kaoriya.net/software/vim/)
- プログラミング関連
  - Git for Windows
  - SourceTree または GitKraken
  - [Cmder](../Cmder/CmderSetup.md)
  - Java JDK
- Diffツール
  - WinMerge
- DB開発ツール
  - A5-Mk2
  - DBeaver
  - pgAdmin
  - SSMS
- パッケージ管理
  - Scoop
- SSH関連
  - PuTTY
  - TeraTerm
- 圧縮ツール
  - 7Zip
  - Explzh
- エクスプローラー拡張
  - QTTabBar
- コマンド拡張
  - PureText
